# Spichr — Pantry & Food Inventory

**A smarter way to manage your kitchen.**  
Track expiry dates, reduce food waste, and stay organized — with iCloud sync, widgets, and Siri Shortcuts.

## Features

- 🔔 **Expiry Notifications** — Custom reminders before items expire
- ☁️ **iCloud Sync** — Seamless sync across devices via CloudKit
- 📱 **Home Screen Widgets** — Glanceable expiry alerts with WidgetKit
- 🛒 **Shopping Lists** — Build lists directly from your inventory
- 🎙️ **Siri Shortcuts** — Voice-driven actions via App Intents
- 📤 **JSON Export** — Full data backup at any time
- 🔍 **Advanced Filtering** — By category, location, expiry, or tags
- 🍳 **Recipe Suggestions** — Based on what you actually have
- 🌍 **18 Languages** — Fully localized
